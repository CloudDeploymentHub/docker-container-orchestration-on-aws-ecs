**Deploying Node.js Application on AWS ECS Fargate (Docker + ECR + ECS + ALB)**

This project demonstrates how to containerize a Node.js application using Docker and deploy it to Amazon Elastic Container Service (ECS) with AWS Fargate, a fully serverless compute engine.
Designed as a real-world cloud deployment example, this repository highlights skills relevant for DevOps engineers with 3–4 years of experience.

**Project Features**

✔ Dockerized Node.js application

✔ Push Docker image to Amazon ECR 

✔ Deploy container to AWS ECS (Fargate – serverless)

✔ Load Balancer integration

✔ Auto scaling configuration

✔ Health checks and monitoring using CloudWatch

✔ Rolling updates without downtime

✔ Complete infrastructure workflow
